## Tor-Relay-Check
Check fasted Tor Relays

## Works only if you are running a tor relay on your Linux machine.

## First install:

```
sudo apt-get install geoip-bin

```

## Then run the script and change the country according to your will.

```
./tor_speed.sh

```
